HTML

COMMON TAGS

TAGS DESCRIPTION
!DOCTYPE Tells the browser what type of file it is opening
html Can only be used once per page and all HTML tags must be within
head Information about the page that the browser uses but is not shown to the user
body All of the content that is displayed in the browser goes within the body tag
h1 - h6 Used for titles on a page, h1 is the largest and used for main titles, size descends and goes to h6 max
p Used for displaying text on a page
a Used for creating links nest a href inside to tell link where to go
div Creates a container
br Ceates a break, all content after will be on a new line
span An inline container, can be used to style specific bits of text with CSS
strong Used to highlight important text, browsers usually render as bold
em Used to emphasize text with italics
sup Inline element used to raise text half a character, often used for footnotes like WWW[1]
sub Same as sup but lowers text instead like H2O
img Used to embed an image
video Embeds a media player for video playback
ul Creates an unordered list with bulletpoints
ol Creates an ordered list with numbers
li Items in a list

SPECICIFITY

ID's have the highest specicifity priority
The order of specicifity is:

  1. ID
  2. Class
  3. Tag

Because of the priority you can use id's to differentiate between things within the same class e.g:


<id="one" class="box">
<id="two" class="box">
<id="three" class="box">

TABLES

TAG DESCRIPTION
table Lables the container as a table
thead Heading of the table i.e titles of columns
tbody Main content of the table
tfoot Summary of the body usually totals
tr Creates a new row in the table
th Lables the content within th tag as a heading
td Data for the cells of the table, each one nested in the same tr is a different column

IMAGES/VIDEOS/AUDIO

Images

To embed an Image use the following syntax:

	<img src="IMAGE SOURCE" alt="A DESCRIPTION OF THE IMAGE">

Videos

To embed a Video use the following syntax:

	<video controls src="VIDEO SOURCE">Text that is shown when video could not load</video>

Audio

To embed audio use the following syntax:

	<audio src="AUDIO SOURCE" controls></audio>

The controls attribute enables the browsers default playback controls on the element.

CSS

Use commas to target multiple elements works for id's and classes: h1, h2, .section, #content

to edit element just type element name on it's own

to target classes use .

to target id use #
